# Security Policy

## Reporting a Vulnerability

NVIDIA is dedicated to the security of our products and services. If you discover a potential security vulnerability in this project, we ask that you report it responsibly.

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Instead, report vulnerabilities through NVIDIA's official channels:

- **NVIDIA Product Security**: https://www.nvidia.com/en-us/security/
- **Email**: psirt@nvidia.com

Please include as much of the following information as possible:

- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested mitigations

NVIDIA will acknowledge receipt and work with you to address verified vulnerabilities.
